	•	計畫概述
	•	計畫緣起與目標
　　本計畫以「公有空間活化」為核心主軸，結合地方創生理念與社會設計方法，重新詮釋新竹縣新豐鄉之地方品牌發展策略。相較於過往以產品或單一產業為導向的地方創生模式，本計畫進一步將「空間」視為產業與人流的關鍵載體，透過閒置或低度使用之公有空間再生，導入青年進駐與多元產業機能，形塑具持續營運能力的地方創生場域，進而帶動在地產業轉型與區域整體活化。
　　本計畫參酌日本「一村一品」運動之精神，強調地方自主性與特色產業發展，同時融合社會設計「以人為本、共創參與」之核心概念，從新豐在地歷史脈絡、產業結構與生活文化出發，透過空間作為介面，建立「人與場域」、「產業與生活」、「文化與體驗」之連結，打造屬於新豐的地方創生品牌體系。
　　在整體架構上，本計畫將原有「人、智、物、聚、景」五大構面，轉化為以「空間活化」為核心的五大發展策略，並透過青年進駐作為推動引擎，說明如下：
	•	人｜青年進駐與地方人才培力機制
　　以公有空間為平台，建立「地方創生青年進駐機制」，透過徵選、培力與陪伴輔導，吸引對地方有認同感之青年團隊進駐，成為空間活化的核心行動者。進駐青年不僅為空間帶來營運能量，更透過自身專業（如餐飲、設計、農產加工、文化策展等），形成多元產業進駐的共創場域。
　　同時，透過培訓課程、實務操作與跨域交流，建立地方人才培育系統，讓青年從「進駐者」轉變為「地方經營者」，進一步帶動在地就業機會與人口回流，形成可持續的地方人才循環。

	•	智｜空間營運與地方創生系統建構
　　導入社會設計與營運管理思維，將公有空間由「閒置資產」轉化為「地方創生基地」。透過整體空間策劃，建立包含品牌識別、動線規劃、營運模式及數位工具之整合系統，使空間不僅為展示或活動場所，更具備產業孵化與市場測試功能。
　　此外，結合數位科技與大數據分析（如人流分析、消費行為、社群聲量），建立地方創生資料庫，作為後續產業發展與政策制定的重要依據，讓空間營運具備策略性與可複製性。

	•	物｜在地產業導入與商品開發
　　以空間為載體，導入新豐在地特色產業（如農產、漁產、加工食品、手作工藝等），透過青年團隊進行產品再設計與品牌化，發展具市場競爭力的地方商品。
　　空間內可規劃為「展示＋銷售＋體驗」複合型場域，使產品不再只是販售，而是結合故事、文化與生活情境的體驗商品。同時透過小量試產、快閃市集與品牌測試機制，降低創業風險，逐步培育地方特色產業鏈。

	•	聚｜地方社群連結與公共空間再生
　　公有空間不僅是產業基地，更是地方社群交流的「公共客廳」。透過空間活化，重新建立居民、青年、遊客之間的互動關係，辦理市集活動、工作坊、講座與文化展演，提升地方參與度與凝聚力。
　　同時串聯在地組織（如社區發展協會、農會、學校、商圈等），形成跨域合作網絡，使空間成為地方治理與公共參與的重要節點，強化地方自主發展能力。

	•	景｜場域再造與地方品牌意象建構
　　透過空間設計與環境改造，重新塑造新豐的地方景觀與品牌形象。從公有市場、閒置建物到周邊街區，導入設計美學與文化敘事，打造具有識別性的創生場域。
　　並進一步串聯周邊觀光資源（如海岸線、濕地、生態景點與在地店家），形成「創生廊帶」，讓空間不只是單點，而是帶動整體區域的觀光與產業發展，提升新豐在區域中的能見度與吸引力。

	•	整體發展策略：從空間出發的地方創生模式
　　本計畫的核心不在於單一活動或短期成果，而是建立一套「以空間為核心、以青年為引擎、以產業為目標」的地方創生運作模式。透過公有空間活化，逐步導入人流、產業與資源，形成正向循環：
空間活化 → 青年進駐 → 產業導入 → 社群聚集 → 品牌形成 → 再帶動空間價值提升
　　最終目標在於協助新豐鄉打造具有在地特色與永續營運能力的地方品牌，使公有空間從「閒置成本」轉化為「創生資產」，並建立可複製、可擴散的地方創生模式。

	•	計畫執行項目
	•	第一階段（115年6月至12月）
本計畫以「公有空間活化基礎建構」為核心，著重於建立品牌能見度、培養地方創生能量及導入專業資源，透過數位行銷、人才培育、產業輔導及文化策展等多元策略，逐步形塑新豐鄉地方創生發展基礎，並為後續產業深化發展與空間營運奠定穩固基礎。
	•	數位行銷與品牌經營（FB社群經營與內容建置）
　　為提升新豐地方創生品牌能見度，本計畫將建立並經營官方Facebook社群平台，作為對外主要資訊傳播與品牌溝通管道。預計每月固定發佈至少2則貼文，內容規劃將以「在地人物故事」、「產業特色介紹」、「空間活化進度」、「活動宣傳」及「文化內容轉譯」等為主軸，透過圖文、短影音及故事敘事形式，提升社群互動與關注度。
　　另將逐步建立內容資料庫，包含影像素材、品牌故事、職人紀錄及地方文化資料，作為未來數位行銷與品牌延伸之基礎。同時，透過數據分析（如觸及率、互動率及粉絲成長）進行滾動式優化，提升社群經營效益，並培養在地團隊具備基本數位行銷能力。
	•	專業輔導與顧問機制導入（至少3場次）
　　為強化地方創生執行能量，本階段將邀請跨領域專業講師與顧問（如品牌設計、空間營運、行銷策略、地方創生實務及社會設計等）進行至少3場次以上之專業諮詢與輔導。
　　輔導內容將採「一對一＋小組共學」方式進行，針對新豐在地團隊與潛力青年進行診斷分析，包含品牌定位、產品優化、通路拓展、空間經營模式及市場策略等，提供具體可行之建議方案。同時，建立顧問陪伴機制，使團隊在計畫期間能持續獲得專業支持，提升其創業與營運能力。
　　透過顧問導入，逐步建立新豐地方創生輔導模型，形成可複製與擴散的地方創生操作機制。
	•	人才培育與課程工作坊（辦理1場以上，至少20人以上）
　　本計畫將以「地方創生人才培力」為核心，規劃辦理至少3場主題課程或工作坊，每場參與人數至少20人，課程內容將結合地方需求與產業發展方向，包含以下主題：
	•	地方創生概念與實務操作
	•	品牌建立與產品設計
	•	社群經營與數位行銷
	•	體驗經濟與活動設計
	•	公有空間營運模式
　　課程將採「理論＋實作」並行方式，透過案例分享、分組討論及實際操作，提升學員學習效果。同時，優先邀請在地青年、返鄉創業者及社區組織參與，逐步培養地方創生種子人才。
　　此外，課程成果將結合後續空間活化與活動策展，使學習成果能實際轉化為行動方案，提升整體計畫實踐性。
	•	新豐鄉地方創生故事展（文化策展與空間活化示範）
　　為強化地方認同與文化傳承，本計畫將規劃辦理「新豐鄉地方創生故事展」，透過策展形式呈現新豐在地歷史、人文、產業與人物故事，作為公有空間活化之重要示範場域。
展覽內容將涵蓋以下主題：
	•	在地職人與產業故事
	•	新豐歷史與文化脈絡
	•	青年返鄉與創生案例
	•	地方創生未來願景
　　展覽形式將結合平面展示、影像紀錄、互動裝置及導覽解說，提升參觀體驗。同時，搭配開幕活動及導覽解說，吸引地方居民與外來遊客參與，增加人流並活化空間使用。
　　展覽成果亦可轉化為數位內容（如線上展覽或短影音），擴大宣傳效益，並作為後續品牌行銷的重要素材。

一、質化效益
　　本計畫以「公有空間活化」為核心，透過青年進駐、產業導入及品牌建構，預期於社會、產業、空間及治理等面向產生多層次質化效益，說明如下：
	•	促進青年回流與地方人才培育
	•	建立青年進駐與創業支持機制，吸引具創意與專業能力之青年投入地方發展
	•	透過課程、輔導及實務操作，培養地方創生所需之跨域人才（行銷、設計、營運等）
	•	提供青年實際營運場域，降低創業門檻與風險，提升留鄉或返鄉意願
	•	建構「地方創生人才循環系統」，使青年由參與者轉為經營者
	•	強化地方認同與社區凝聚力
	•	透過故事展、活動及策展，重新詮釋新豐在地文化與歷史
	•	促進居民、青年與外來參與者之交流互動，形成共同參與氛圍
	•	強化地方居民對公有空間之認同感，使其成為生活與交流核心
	•	建立「公共參與」與「共創治理」模式，提升社區自主發展能力
	•	推動地方產業轉型與價值提升
	•	協助在地產業由傳統銷售模式轉型為品牌化與體驗化經營
	•	透過設計與行銷導入，提高產品附加價值與市場辨識度
	•	結合AI行銷與數位工具，提升地方產業競爭力
	•	導入ESG理念，促進產業朝向永續與環境友善發展
	•	建構地方品牌與整體形象
	•	透過系統性品牌策略，建立新豐鄉整體品牌識別
	•	強化「人（青年）」、「物（產品）」、「場（空間）」之整合行銷
	•	建立地方品牌故事，使其具備文化深度與情感連結
	•	提升新豐在區域與全國層級之辨識度與吸引力
	•	活化公有空間並提升使用價值
	•	將閒置或低度使用之公有空間轉化為創生基地
	•	建立多功能使用模式（展售、體驗、交流、創業）
	•	提升空間使用頻率與多樣性，增加公共資產效益
	•	透過空間營運，帶動周邊區域發展與人流聚集
	•	促進跨域合作與資源整合
	•	串聯政府、民間團體、企業與學術資源
	•	建立跨領域合作平台（設計、農業、文化、科技）
	•	媒合企業CSR與ESG資源，導入外部支持
	•	形成地方創生合作網絡，提升整體發展能量
	•	建立可持續營運之地方創生模式
	•	建立「空間×產業×品牌×營運」整合模式
	•	透過分潤與商業機制，提升計畫自償能力
	•	建構創生循環（人流→消費→收益→再投入）
